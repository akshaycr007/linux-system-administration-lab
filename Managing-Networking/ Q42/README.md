# Q42 - Configure Network

## Objective
Configure networking between VM1, VM2, VM3, Host PC, Mobile/Laptop and Internet.

## Environment

VM1: 192.168.50.176
VM2: 192.168.50.173
VM3: 192.168.50.172

Network Mode: Bridged Adapter

## Verification

- VM1 can ping VM2
- VM1 can ping VM3
- VM2 can ping VM3
- Host PC can communicate with VMs
- VMs can access Internet
- Mobile/Laptop can access VMs

## Result

Network communication verified successfully.
